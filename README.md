# Reconhecimento Facial com Python – DeepFace + Câmeras IP

Este repositório apresenta um sistema de **reconhecimento facial em tempo real**, desenvolvido em Python, com integração à biblioteca **DeepFace** e suporte a **câmeras IP**. A solução é voltada para aplicações em **segurança, controle de acesso e autenticação automatizada**, podendo ser adaptada a diversos ambientes institucionais ou corporativos.

## Objetivo

Implementar um sistema funcional de reconhecimento facial com alta precisão, utilizando inteligência artificial, visão computacional e streaming de vídeo por IP, visando identificar indivíduos com base em comparações de embeddings faciais armazenados.

## Funcionalidades

- Captura de vídeo ao vivo via câmera IP;
- Reconhecimento facial com a biblioteca **DeepFace**;
- Comparação com base de dados local de embeddings;
- Alertas e registros em tempo real;
- Modularidade para expansão futura com logs, banco de dados e notificações.

## Tecnologias Utilizadas

- **Python 3.x**
- **OpenCV** para captura de vídeo e processamento de imagem
- **DeepFace** para análise facial e geração de embeddings
- **NumPy**, **MySQL**, **pickle**, entre outras bibliotecas auxiliares
- Suporte a **fluxo de vídeo via IP** e possíveis ajustes para ambientes com múltiplas câmeras

## Requisitos

- Python instalado e ambiente virtual configurado
- Câmera IP funcional com endereço de streaming acessível
- Dependências listadas no arquivo `requirements.txt`
- Embeddings cadastrados previamente no modelo

## Aplicações Reais

- Controle de entrada em laboratórios e salas restritas;
- Registro de presença automatizado;
- Sistemas experimentais de autenticação facial;
- Ambientes educacionais, institucionais ou empresariais.
